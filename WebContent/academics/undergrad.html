Undergraduate Studies

I completed my Bachelor's in Science and Engineering Computer Science Engineering in 2007, with a minor in Mathematics. As an undergrad at the University of Pennsylvania I was very interested in Compilers, and then some Computer Architecture after CSE 240/371, until my internship at Accenture. That got me interested in Computer Security and I was able to parlay that into a cool, to Gerry and I at least, Senior Design Project. It boiled down to creating a program that translates a specially annotated Java source code into Java Smart Card and terminal code which would allow the Smart Card program to send a request to the terminal, and wait for a response. The full paper is available here: Refactoring the Java Card Framework.
